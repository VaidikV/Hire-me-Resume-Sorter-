## Libraries needed:
```
pip install pdfminer        # python 2
pip install pdfminer.six    # python 3
pip install spacy
python -m spacy download en_core_web_sm
pip install nltk
python -m nltk nltk.download('words')
pip install tkPDFViewer
```
